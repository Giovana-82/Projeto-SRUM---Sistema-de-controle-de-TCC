# Sistema de Controle de TCC

## Descrição do Projeto

O Sistema de Controle de TCC é uma aplicação web proposta para auxiliar no gerenciamento de Trabalhos de Conclusão de Curso (TCC). O sistema permite o cadastro de alunos, orientadores, trabalhos e entregas, facilitando o acompanhamento do progresso de cada projeto acadêmico.

Este projeto foi desenvolvido como uma simulação de uma Sprint utilizando a metodologia Scrum na disciplina de Engenharia de Software.

## Objetivo

Desenvolver uma solução capaz de:

- Cadastrar alunos;
- Cadastrar orientadores;
- Registrar trabalhos de conclusão de curso;
- Registrar entregas parciais e finais;
- Acompanhar o andamento dos TCCs;
- Visualizar o status de cada trabalho.

## Funcionalidades

### Gerenciamento de Alunos

- Cadastro de alunos;
- Consulta de alunos cadastrados.

### Gerenciamento de Orientadores

- Cadastro de orientadores;
- Consulta de orientadores cadastrados.

### Gerenciamento de TCCs

- Cadastro de trabalhos;
- Associação entre aluno e orientador;
- Registro de status do trabalho.

### Acompanhamento

- Registro de entregas;
- Consulta do andamento do projeto;
- Dashboard com informações gerais.
